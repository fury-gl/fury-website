fury.io.load_image(filename, as_vtktype=False, use_pillow=True)[source]¶

Load an image.

Parameters
filename: str

should be png, bmp, jpeg or jpg files

as_vtktype: bool, optional

if True, return vtk output otherwise an ndarray. Default False.
